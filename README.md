# FUTURE_DS_01 - Business Sales Performance Dashboard

## Overview
This project analyzes business sales performance using Power BI.

## Tools Used
- Power BI
- Excel/CSV Dataset

## Dashboard Features
- Sales Trend Analysis
- Sales by Category
- Profit by Region
- Sales by Segment
- Product Performance Analysis

## Key Findings
- Technology category generated the highest sales.
- West region produced the highest profit.
- Consumer segment contributed the largest share of sales.
